Artificially Intelligent Virtual Assistant

AIVA - Artificially Intelligent Virtual Assistant - is
a personal assistant designed to handle basic
tasks via talk or text. The home window pictured
to the right provides access to textual searches
as well as a menu bar via which the user can
access proprietary settings Upon launch, AIVA
will display a travel animation bringing the
window to the upper-right portion of the screen.

The user can also access the theme settings of
AIVA with default options of Light - a combination
of blue and white - or Dark - a combination of red
and black as pictured to the left. By default the
theme is set to dark. In the future the user will
be able to add custom themes to further
personalize his or her assistant. The menu bar
also contains access to personalization and
preferences settings that specify aspects of
both the windows and AIVA's responses.

In addition to theme customization, the user
has the ability to add/change settings such as
username, name, password, age, email, and
phone number. When such actions are
performed via text search (ex: "change my name
to Craig"), a window appears to notify the
user that a change has been made. The goal
with these settings is to eventually give
AIVA to ability to send notifications
directly to the user's mobile devices.

Should the user choose to do so, he or she
can disable User Save mode. Upon this action,
the prompt pictured to the left will appear
each time AIVA is launched. The login window
will be displayed with the default theme unless
the user chooses to change the theme prior to
the most recent exit. In this case, the window
will be displayed with the most recent theme.
