About

AIVA is a personal assistant designed to handle basic tasks via talk or text. The home window provides access to textual searches - internet and local - as well as a menu bar via which the user can access settings. When the assistant opens for the first time, a login window is displayed with a default username and password of "root." However, a "remember me" setting is selected by default and the user's credentials can be changed through either the settings menu or the search bar.
