AIVA is an artificially intelligent virtual assistant designed to handle basic tasks via talk or text. The home window provides access to textual searches - Internet or local - as well as a menu bar by which the user can access settings. When the assistant opens for the first time, a login window is displayed with a default username and password of "root." However, a "remember me" setting is selected by default and the user's credentials can be changed either through the settings menu or the search bar.
