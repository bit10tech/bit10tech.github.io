Wilson's Theorem

Wilson's Theorem - actually proved by
Joseph-Louis Lagrange in 1770 - is a theorem
proposed by John Wilson in the 18th century
that determines whether or not a number is prime.
The theorem states that if the factorial of a
number in the modulo of that number equals -1
then the number is prime. For example, 2, a known
prime number, satisfies this theorem because the
factorial of 2-1 is 1, which is 1 mod 2, and is
thus equal to -1 mod 2, which is 1. The code to the
right demonstrates this theorem by taking the input
of any integer greater than 0 and using Wilson's
Theorem to determine if the number is prime.
